## 使用说明

表情包文件结构如下

```
├── 表情包主目录
│   ├── 表情包文件夹1
│   │   ├── 图片1.png
│   │   ├── 图片2.png
│   │   ├── ...
│   ├── 表情包文件夹2
│   │   ├── 图片1.png
│   │   ├── 图片2.png
│   │   ├── ...
│   ├── ...
```

支持的图片类型
 - .jpg
 - .jpeg
 - .png
 - .gif
 - .bmp
 - .webp

### 注意事项
- 图片名称不能包含特殊字符，如 `%}{&}` 等

## 开发 && 打包
运行 `wails dev` 命令启动项目

运行 `wails build` 命令打包项目
